
# smart-doc <small> ![maven](https://img.shields.io/maven-central/v/com.github.shalousun/smart-doc) </small>


> 一个无侵入式的api接口文档生成器.

* `smart-doc`是一款同时支持`JAVA RESTful API`和`Apache Dubbo RPC`接口文档生成的工具。
* 完全基于注释生成文档，做到零侵入。
